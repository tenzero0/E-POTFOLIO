Neon Effects

Create glowing UI elements with CSS shadows and transitions.

.neon-btn {
  padding: 12px 24px;
  border: 2px solid #00ffcc;
  background: transparent;
  color: #00ffcc;
  font-weight: bold;
  border-radius: 8px;
  transition: 0.3s;
}

.neon-btn:hover {
  background: #00ffcc;
  color: #111;
  box-shadow: 0 0 15px #00ffcc, 0 0 30px #00ffcc;
}

Animations

Create smooth animations and transitions with CSS keyframes.

.animation-example {
  width: 100px;
  height: 100px;
  background: rgba(0, 255, 200, 0.2);
  border-radius: 50%;
  animation: pulse 2s infinite;
}

@keyframes pulse {
  0% {
    transform: scale(0.9);
    box-shadow: 0 0 0 0 rgba(0, 255, 200, 0.7);
  }
  70% {
    transform: scale(1);
    box-shadow: 0 0 0 15px rgba(0, 255, 200, 0);
  }
  100% {
    transform: scale(0.9);
    box-shadow: 0 0 0 0 rgba(0, 255, 200, 0);
  }
}
